### ";" and alert
```js
function("message"); // ; is the end
```

[Alert](https://developer.mozilla.org/en-US/docs/Web/API/Window/alert):  Pops out an alert window.
```js
alert("string");
```
### 3 types of data

A way of classifying data.

- **String**: text.
- **Numbers**: 1, 2, 3, 4.
- **Boolean**: true, false.

[typeof()](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Operators/typeof):  returns a string indicating the type of the operand's value
```js
typeof();
```
